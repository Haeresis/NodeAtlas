Namespace: webconfig

NA. webconfig

Contain Webconfig file to JSON format.

Members

staticNA.webconfig.assetsRelativePathstring

Folder for public file like image, CSS or JS.

Default Value:
  • "assets/".

staticNA.webconfig.autoGenerateboolean

Allow NodeAtlas to generate real file into NA.webconfig.generatesRelativePath directory if set to true.

Default Value:
  • false.

staticNA.webconfig.bundlesObject

Contain bundle configuration for CSS and JS minification to JSON format.

Properties:
Name Type Description
bundles Object

The Bundles object.

Properties
Name Type Description
javascript Object

The Bundles configuration for Javascript.

Properties
Name Type Description
files Object

Each object name represent an output file and each property of object represent an input file.
enable boolean

No JavaScript minification if set to false.
stylesheets Object

The Bundles configuration for CSS.

Properties
Name Type Description
files Object

Each object name represent an output file and each property of object represent an input file.
enable boolean

No CSS minification if set to false.
Example
{
   "javascript": {
       "files": {
           "javascript/framework.min.js": [
               "javascript/modernizr.js",
               "javascript/jquery.js",
               "javascript/prettify.js",
               "javascript/prettify/run_prettify.js"
           ],
           "javascript/common.min.js": [
               "javascript/components/extended-format-date.js",
               "javascript/common.js"
           ]
       }
   },
   "stylesheets": {
       "files": {
           "stylesheets/common.min.css": [
               "stylesheets/common.css",
               "stylesheets/common-min780.css",
               "stylesheets/common-min1160.css"
           ]
       }
   }
}

staticNA.webconfig.commonVariationstring

Name of file for common variation.

staticNA.webconfig.componentsRelativePathstring

Default folder for template engine include file or controller components.

Default Value:
  • "components/".

staticNA.webconfig.controllersRelativePathstring

Controller folder for Back-end.

Default Value:
  • "controllers/".

staticNA.webconfig.generatesRelativePathstring

HTML asset generation Folder.

Default Value:
  • "generates/".

staticNA.webconfig.getSupportboolean

Allow you to avoid or authorize GET response for all page.

Default Value:
  • true

staticNA.webconfig.httpHostnamestring

Server listening hostname by http (for reverse proxy).

Default Value:
  • "localhost", or `process.env.IP_ADDRESS` if setted, or the webconfig's property `httpHostname`.

staticNA.webconfig.httpPortstring

Server listening port.

Default Value:
  • 80, or `process.env.PORT` if setted, or the webconfig's property `httpPort`.

staticNA.webconfig.httpSecureboolean

Define is site is running with HTTP(S) protocol.

staticNA.webconfig.indexPageboolean

Allow NodeAtlas to create a root page with link to all routes for development if set to true.

Default Value:
  • false.

staticNA.webconfig.languageCodestring

Represent the global and main language code for website.

Default Value:
  • No language code if not setted.

staticNA.webconfig.pageNotFoundstring

Represent route to use if no route match in all route.

staticNA.webconfig.postSupportboolean

Allow you to avoid or authorize POST response for all page.

Default Value:
  • true

staticNA.webconfig.routesObject

Contain all routes to JSON format.

Default Value:
  • The webconfig's property `routes`.

staticNA.webconfig.sessionObject

Use a more complexe session cookie options. Replace NA.webconfig.sessionKey and NA.webconfig.sessionSecret if set.

See:
  • Session Middleware

staticNA.webconfig.sessionKeystring

Name for Session cookie of connected user.

Default Value:
  • 'nodeatlas.sid'.

staticNA.webconfig.sessionSecretstring

Secret for Session cookie of connected user.

Default Value:
  • '1234567890bépo'.

staticNA.webconfig.templateEngineClosePatternstring

Set close bracket for use EJS.

Default Value:
  • '%>'.

staticNA.webconfig.templateEngineOpenPatternstring

Set open bracket for use EJS.

Default Value:
  • '<%'.

staticNA.webconfig.templatesRelativePathstring

Template folder for template engine.

Default Value:
  • "templates/".

staticNA.webconfig.urlHostnamestring

Url access hostname by http (for reverse proxy).

Default Value:
  • `NA.webconfig.httpHostname`.

staticNA.webconfig.urlPortstring

Url access port.

Default Value:
  • `NA.webconfig.httpPort`.

staticNA.webconfig.urlRelativeSubPathstring

Adding subfolder to original url.

Default Value:
  • Empty.
Example
// If `NA.webconfig.urlRelativeSubPath` is setted to "example/"
// Website will run by default to « http://localhost/example/ »

staticNA.webconfig.urlWithoutFileNamestring

Website http(s) url generate depending of NA.webconfig.httpSecure, NA.webconfig.urlHostname and NA.webconfig.urlPort. This value does not contain NA.webconfig.urlRelativeSubPath.

staticNA.webconfig.variationsRelativePathstring

Language and variable variation folder depending of languages.

Default Value:
  • "variations/".
