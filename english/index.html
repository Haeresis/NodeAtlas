CLI Usage

Install NodeAtlas

> npm install -g node-atlas

Create test website

> nodeatlas --init

Run website

> nodeatlas --browse

API Usage

Install NodeAtlas

> npm install node-atlas

Create a script

server.jsvar NA = require("node-atlas"),
    website = NA();
website.config({
    "init": true
}).afterInitProject(function () {
    website.run({
        "browse": true
    });
}).init();

Run website

> node server.js
